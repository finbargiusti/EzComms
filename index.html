Quick start

Clients are easy

#include "EzComms.h"
#include <iostream>

int main() {
	EzComms::setupConn(client, 127.0.0.1, 1337);
	EzComms::Socket conn;
	std::cout << conn.recv() << std::enddl;
}

Servers are too

#include "EzComms.h"
#include <iostream> 

int main() {
	EzComms::setupConn(server, 127.0.0.1, 1337);
	EzComms::Socket conn;
	getline(std::cin, const char *stdin);
	conn.send(stdin, (uint32_t)stdin.size())
}

And they work magnificently:

 > make client.cpp -o client EzComms.cpp
 > make client.cpp -o client EzComms.cpp
 > ./server                   | > ./client                  
 Hello world                  | Hello world                 
 >

Setup Connection

void EzComms::setupConn(enum connectionType {client, server}, int ip_address, int port);

 This function is used to set up the application for the connection. You pass the connection type (server or client), the IP to connect to or listen on, and the port to connect or bind to.
