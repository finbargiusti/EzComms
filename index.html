Examples

It really is easy, this is ALL you need for a client that connects to a server, receives data and prints it.

#include <iostream>
#include "EzComms.h"

int main() {
	EzComms::setupConn(client);
	EzComms::Socket pipe;
	printf("%s\n", pipe.recv().c_str());
}

Step-by-step tutorial

First, you have to include the library

#include "EzComms.h"

Next, you have to set up the app for connections as such:

int main () {
	EzComms::setupConn(client); // If you are writing the client
	EzComms::setupConn(server); // If you are writing the server
}
